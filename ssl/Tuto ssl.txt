Pour générer des certificats autoSignés:
(voir tuto ici: http://www.hacksparrow.com/node-js-https-ssl-certificate.html )

Ouvrir openSSl puis

// Générer la clef privée:
openssl> genrsa -out nom-quelquonque-key.pem 1024 

// Création d'une CRS (Requête de signature):
openssl> req -new -key nom-quelquonque-key.pem -out certrequest.csr
... bunch of prompts

// Création d'un certificat auto-signé à partir de la CSR:
openssl> x509 -req -in certrequest.csr -signkey nom-quelquonque-key.pem -out nom-quelquonque-cert.pem

Ca vas générer 3 fichiers:
- nom-quelquonque-key.pem 
- nom-quelquonque-crt.pem 
- certrequest.csr
-----------------------------------------
Pour générer des certificats autoSignés version 2:
( source: http://blog.georgio.fr/comment-creer-un-certificat-ssl/)

# Création de la clé privée :
-----------------------------
#  genrsa : Créer une clé privée de type RSA
#  -des3 : Utiliser l'option de cryptage (ciphers) DES3
#  -out server.key : Fichier de sortie (nom de la clé privée)
#  1024 : Taille de la clé privée (par défaut : 512)

openssl genrsa -des3 -out server.key 1024
*On vous demandera de choisir une pass-phrase (mot de passe), à conserver soigneusement.

# Création d'une CSR (Requête de signature) pour générer le certificat :
------------------------------------------------------------------------
#  req : On dit que cela concerne une CSR
?#  -new : C'est pour une création (de CSR...)
#  -key : Avec la clé privée "server.key"
#  -out : Fichier de sortie (nom de la CSR)

openssl req -new -key server.key -out server.csr

*On vous demandera plusieurs infos personnelles ainsi 
que le nom de domaine pour lequel est dédié ce certificat. 
Ce système permet d'envoyer une CSR à une AC afin 
qu'elle puisse la signer sans avoir à connaître votre clé privée.

# Création d'un certificat auto-signé à partir de la CSR:
----------------------------------------------
# x509 : On informe que cela concerne un CRT
# -req : C'est pour soumettre une création
# -days 365 : Pour une date de validité de 1 an
# -in server.csr : Avec "server.csr" pour CSR
# -signkey server.key : Signer avec notre propre clé privée
# -out server.crt : Fichier de sortie (nom de la CRT)

openssl x509 -req -days 365 -in server.csr -signkey server.key -out server.crt

*Afin de pouvoir signer avec notre propre clé privée, 
il vous sera demandé la passphrase que vous avez renseigné 
lors de la création de la clé privée.
