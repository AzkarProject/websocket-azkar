Fakerobubox: software emulation of an embedded robot system for mapping, battery gauge and drive controls.
If enabled, ignores the loaded map. The simulation is based on the I3S laboratory map (Labo261.png).
Automatic Points of interest navigation and relocation commands are disabled in this mode.
You can use it to simulate a robot client in a tab an a pilot client in other tab
